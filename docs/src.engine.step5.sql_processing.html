src.engine.step5.sql_processing module¶

src.engine.step5.sql_processing.render_template_query(config, general_query, args_dict)[source]¶

Parameters

  • config (module) – General tool configuration.

  • general_query (str) – General SQL query.

  • args_dict (dict) – Dictionary of processed arguments from step 2.

Returns
